# 12 – Glossar

| Begriff / Abkürzung   | Bedeutung / Erklärung                                                                                                                    |
|-----------------------|------------------------------------------------------------------------------------------------------------------------------------------|
| **WCAG 2.2**          | Web Content Accessibility Guidelines ist eine internationale Richtlinien zur Barrierefreiheit von Webinhalten.                           |
| **Program.cs**        | Zentrale C#-Konfigurationsdatei.                                                                                                         |
| **csproj**            | Projektdatei eines .NET-Projekts, die Abhängigkeiten, Build-Einstellungen und Zielplattformen definiert.                                 |
| **SQLite**            | Leichtgewichtige, dateibasierte SQL-Datenbank; standard bei dem Template das für die Webseite verwendet Wurde                            |
| **Docker**            | Zentrale Plattform zur Containerisierung von Anwendungen. Erleichtert Build und Deployprozesse                                           |
| **AntiForgeryTokens** | Verhindern Cross Site Request Forgery Attacken.                                                                                          |
| **Razor Pages**       | Seitenbasiertes Webkonzept in ASP.NET Core, das C#-Logik und HTML-Markup in einer strukturierten Seite kombiniert.                       |
| **Blazor Component**  | Wiederverwendbare UI-Komponente in Blazor, bestehend aus C#-Logik und Razor-Markup, die client- oder serverseitig gerendert werden kann. |
| **Service**           | Klasse zur Kapselung von Geschäftslogik oder wiederverwendbaren Funktionen, die über Dependency Injection in der Anwendung genutzt wird. |
| **Entity**            | Objekt, das eine Datenbanktabelle repräsentiert und typischerweise mit einem ORM wie Entity Framework verwendet wird.                    |
